## Commit 로그 규칙

- 포멧 : `{prefix}: {변경 내용} (#{이슈번호})`
- 예시 : `feat: Dropbox 컨텐츠 연동 (#130)`
- prefix
  - 기능 추가 : `feat:`
  - 오류 수정 : `fix:`
  - 잡다한 일 : `chore`
  - 문서화 : `doc:`
  - 리팩토링 : `refactor:`
